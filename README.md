# todo

## Issue Tracker

### Essential Features

* Actions: Create Issues
* Fields: Description, Status

#### Creating an issue
todo --file todo.dat new create nice UI
> todo created: 1

todo --file todo.dat list
> 1 - create nice UI

todo --file todo.dat done 1
> 1 marked as done

### Nice to have

* If case sensitive key search fails, try case insensitive ("Did you mean...?")
